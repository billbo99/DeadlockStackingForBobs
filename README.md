# Deadlock Stacked Recipes

Takes deadlock stacked items and combines them to create deadlock stacked items

EG stacked iron + stacked copper cable to make stacked green chip


## Basic testing with
* Space Exploration
* Bobs
* Angels
* Pyanadon
* Omnimatter
* Yuoki
* Krastorio Legacy
